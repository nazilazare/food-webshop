# food-webshop
A sustainable food delivery service in Norway connects customers with fresh products from local farms. Customers can browse and order groceries online, with deliveries made directly to their homes from partner farms.

## Run locally

Windows PowerShell:

```powershell
npm start
```

Then open http://localhost:8080.

If port 8080 is busy:

```powershell
$env:PORT=3000
npm start
```

## Alternate (VS Code Live Server)
- Open `index.html`
- Right‑click → Open with Live Server
